## Job Description (inferred via WebSearch)

Verkada builds cloud-managed physical security systems — cameras, access control, environmental sensors — unified on a single platform. The AI Software Engineer Intern role sits on the ML/computer vision team building real-time inference pipelines that process video and sensor streams from 100k+ deployed devices.

**Responsibilities:**
- Design and implement ML models for computer vision tasks (object detection, classification, tracking) in production
- Optimize model inference for edge deployment and cloud-side processing
- Develop and maintain data pipelines for model training and evaluation
- Collaborate with hardware and firmware teams to integrate AI capabilities into physical security devices
- Evaluate multimodal AI approaches combining video, audio, and sensor data

**Preferred Qualifications:**
- Strong Python and PyTorch experience; familiarity with TensorFlow
- Computer vision background: object detection, segmentation, tracking
- Experience with model training pipelines and evaluation metrics (mAP, AP50, etc.)
- Familiarity with ML infrastructure: data pipelines, annotation workflows, model registries
- Knowledge of edge/embedded inference optimization (ONNX, TensorRT, quantization) a plus
- Experience with OpenCV, NumPy, pandas for data processing

**Keywords:** computer vision, PyTorch, Python, ML inference, object detection, segmentation, multimodal AI, video analysis, sensor data, production ML, data pipeline, model training, AP metrics, COCO, YOLOv8
