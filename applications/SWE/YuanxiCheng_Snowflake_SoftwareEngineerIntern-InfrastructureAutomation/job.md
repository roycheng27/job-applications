## Job Description (inferred via WebSearch)

Snowflake's Infrastructure Automation team builds the tooling and frameworks that keep Snowflake's globally distributed cloud data platform running reliably at scale. The intern role focuses on automating operational workflows, building internal tooling, and improving observability across Snowflake's infrastructure.

**Responsibilities:**
- Design and implement automation scripts and frameworks in Python or Java to reduce manual operational toil
- Build and maintain internal tooling for infrastructure provisioning, monitoring, and deployment
- Develop data pipelines and queries to surface operational metrics and health signals
- Collaborate with SRE and platform engineering teams to scope automation priorities
- Write clean, testable code with strong documentation

**Preferred Qualifications:**
- Strong Python programming skills; familiarity with scripting and workflow automation
- SQL proficiency — ability to write complex queries against large datasets
- Experience with cloud platforms (GCP, AWS, Azure); familiarity with Snowflake a plus
- Understanding of data pipelines, ETL, and database design
- Exposure to infrastructure tools (Terraform, Kubernetes, CI/CD pipelines) a plus
- Strong data structures and algorithms foundation

**Keywords:** Python, SQL, automation, infrastructure, cloud, Snowflake, BigQuery, data pipelines, ETL, distributed systems, tooling, workflow automation, observability, scripting
