## Job Description (inferred via WebSearch)

Tesla's Service Engineering team builds the software systems that support vehicle diagnostics, repair workflows, and aftermarket service operations. The intern role involves developing software that interfaces with vehicle data systems, service tooling, and internal operational platforms.

**Responsibilities:**
- Develop software for vehicle diagnostics and service tooling using C++ and/or Python
- Build and maintain data processing pipelines for service event data and vehicle telemetry
- Design algorithms and data structures for efficient fault detection and diagnostics workflows
- Collaborate with cross-functional teams (hardware, firmware, service ops) to scope features
- Write clean, performant code with robust testing

**Preferred Qualifications:**
- Strong C++ and/or Python programming skills
- Solid foundation in data structures and algorithms
- Familiarity with embedded systems concepts or vehicle software a plus
- Experience with SQL and database design for operational data
- Experience with data processing pipelines and large-scale datasets
- Ability to work in a fast-paced, cross-functional engineering environment

**Keywords:** C++, Python, data structures, algorithms, diagnostics, service engineering, vehicle software, software systems, SQL, data pipelines, embedded systems, telemetry
