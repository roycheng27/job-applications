## Job Description (inferred via WebSearch)

Remitly is a digital financial services company focused on international money transfers. The Pricing Platform SDE1 role sits at the intersection of data engineering and backend software — building the systems that compute, store, and serve dynamic pricing for millions of transfer transactions globally.

**Responsibilities:**
- Build and maintain the pricing platform backend: APIs, data models, and pricing computation engines in Java and Python
- Design scalable microservices that compute real-time exchange rates and fee structures
- Develop and improve data pipelines that ingest market rate feeds and produce pricing signals
- Write SQL queries and design database schemas for pricing data storage and analytics
- Collaborate with product, data science, and finance teams on pricing strategy implementation

**Preferred Qualifications:**
- Strong Java and/or Python programming skills
- Solid SQL skills — ability to design schemas, write complex queries, and optimize performance
- Experience with cloud infrastructure (AWS preferred, GCP acceptable)
- Understanding of distributed systems, microservices architecture, and API design
- Experience with data pipelines or ETL workflows a plus
- Exposure to financial data, time-series data, or pricing systems a plus
- Strong data structures and algorithms foundation

**Keywords:** Java, Python, SQL, pricing, fintech, distributed systems, microservices, backend engineering, data pipelines, Snowflake, cloud, AWS, data structures, algorithms, ETL
