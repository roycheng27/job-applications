## Job Description (inferred via WebSearch)

Rippling is an all-in-one HR/IT/Finance platform that automates employee lifecycle management for businesses. The Software Engineer Intern role involves building and scaling the core platform used by thousands of companies globally — touching payroll engines, HRIS data systems, integrations, and internal tooling.

**Responsibilities:**
- Build and maintain scalable backend services in Python and/or Java
- Design and implement APIs and data models for HR, payroll, and finance workflows
- Write clean, tested code that ships to production and handles real customer data
- Collaborate with product and design to ship full-stack features end-to-end
- Debug and optimize performance of backend systems serving high-traffic workloads

**Preferred Qualifications:**
- Strong Python and/or Java programming skills
- Solid foundation in data structures, algorithms, and system design
- Experience with SQL and relational databases
- Familiarity with REST APIs, microservices, and backend engineering patterns
- Exposure to distributed systems concepts and cloud infrastructure
- Bonus: experience with React or other frontend frameworks

**Keywords:** Python, Java, SQL, distributed systems, backend engineering, API design, payroll, HR tech, data structures, algorithms, scalability, REST API, microservices
